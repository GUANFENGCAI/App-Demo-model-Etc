This Application is based on javafx and java 17

API key and Imgur client id are environment variables, if you want to run the application in online version, please set your own environment variable on personal device, or you can just run the offline version.

Online Version:
	grade run --args="online online"

Offline Version:
	grade run --args="offline offline"

WeatherBit API:
	https://www.weatherbit.io/api/weather-current

Imgur API:
	https://apidocs.imgur.com/#c85c9dfc-7487-4de2-9ecd-66f727cf3139
